# Vue-cli3

> 按照官网要求，首先卸载之前的版本

## 检查环境

```shell
$ vue -V
# 如果提示有版本，但不是3.x.x，则需要卸载
```

## 卸载老版本

```shell
$ npm uninstall vue-cli
```

## 安装

```shell
# Mac需要sudo权限
$ sudo npm install -g @vue/cli
# 检查版本
$ vue -V
3.2.2
```

# 创建项目

## 图形方式

在`vue-cli3`中增加了图形化的创建方式

```shell
$ vue ui
```

自动在浏览器中运行`http://localhost:8000`，启动项目管理界面

按照提示一步步完成即可
