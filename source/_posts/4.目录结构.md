title: VUE 之目录结构

categories: vue

------

> `Vue-cli3` 在文件组织和目录结构上与之前的版本有了更新。
>
> 以下的目录结构建议在`vue-cli3`创建的项目中使用

```shell
├── design
├── dist
│   ├── css
│   ├── fonts
│   ├── img
│   └── js
├── public
├── src
│   ├── assets
│   ├── components
│   ├── plugins
│   └── views
└── tests
    └── unit

+ dist
	+ css
	+ fonts
	+ img
	+ js
	- favicon.ico
	- index.html
+ public
	- favicon.ico
	- index.html
+ src
	+ `api`
	+ assets
		- logo.png
	+ components
		- *.vue
	+ plugins
		- *.js
	+ views
		- *.vue
	- app.vue
	- main.js
	- router.js
	- store.js
+ tests
- .browserslistrc
- .eslintrc.js
- .gitignore
- babel.config.js
- package-lock.json
- package.json
- postcss.config.js
- README.md
- vue.config.js
```

在此基础上增加一些项目自用目录

```shell
+ src
	+ api
	+ directive
	+ filter
	+ mock
	+ router
	+ styles
	+ utils
```
